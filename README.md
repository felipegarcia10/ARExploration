Instructions:

1. clone the project from the repo
2. open the project in Unity
3. go to the ARExploration scene
4. run the project



This is a prop placement experience in which you can place different props based on plane detection.
The system detects the alignment of the plain you are pointing to and based on that it will spawn a different object:

* if the plane is vertical (wall), it will spawn a painting.
* if the plane is horizontal looking up (table or floor) it will spawn a vase.
* if the plane is horizontal looking down (ceiling) it will spawn a chandelier.



The UI text will also change based on the plane alignment.

you can delete any spawned object by taping it.
